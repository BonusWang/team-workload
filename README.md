# TeamWorkload - 团队工作量管理系统

## 项目介绍

TeamWorkload 是一款面向开发团队的内部工具系统，旨在统一管理团队人力与工作量数据。通过自动统计工作量分布，帮助项目经理识别人力饱和与空缺情况，支撑团队资源合理调配，提升团队管理效率。

**核心功能：**

- 📊 个人/团队本周工作量周报（统计周期：上周四至本周四）
- 📈 个人/团队月度工作量汇总
- 📅 请假管理（登记、直属上级审批、余额查询）
- 🎯 个人工作量可视化看板（含月度日历）
- 👥 团队人力空缺分析与预警
- 📥 Excel 数据导入（全开系统模板/其他投入清单模板）
- 📤 报表导出 Excel

**技术栈：**

| 层级 | 技术 |
| :--- | :--- |
| 前端 | Vue3 + Element Plus + ECharts + Pinia |
| 后端 | Spring Boot 2.x（单体服务，前后端一体） |
| JDK | Corretto 1.8.0_482 |
| 数据库 | MySQL 5.7 |
| 部署 | Docker |
